# HubsFood -Rótulo

> ServiçoHub centraliza catálogos de diversos serviços, facilitando o acesso rápido e prático a ofertas e empresas em um só lugar.

## Problema:  Lugares como praças de alimentação de shoppings ou foodtrucks não tem um qrcode por exemplo que uma os cardápios de todas as lojas locais, o que obriga o cliente a ter que visitar cada loja ou restaurante para assim fazer o pedido. Isso cria uma experiência inconveniente, especialmente em locais movimentados.

A nossa solução é criar uma página protótipo onde cada restaurante (food truck) possa atualizar seus cardápios em tempo real. Além disso, haveria uma página central que reuniria todos os cardápios, permitindo que os clientes, a partir da mesa, possam visualizar as opções de todas as lojas(food trucks) no evento, sem precisar se deslocar.
